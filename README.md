#  BuzzZik – Blind Test Moderne

BuzzZik est un blind test web multijoueur en équipe, rapide, fun et musical.  
Jouable depuis plusieurs téléphones, sans compte, en famille ou entre amis !

##  Fonctionnalités
-  Lecture d’extraits musicaux
-  Jeu en équipes avec avatars
-  Connexion par code de partie
-  Questions à choix multiple
-  Scores, transitions, et podium final

##  Stack technique
- Frontend : React + TailwindCSS
- Backend : C# (.NET)
- Temps réel : SignalR (ou Firebase)
- UI : Mobile-first
